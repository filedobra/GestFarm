# Gestione Farmaci - Applicazione Web

![Anteprima dell'interfaccia](https://via.placeholder.com/800x400.png?text=Gestione+Farmaci+Preview) <!-- Sostituisci con un'immagine reale -->

Un'applicazione web per la gestione completa di farmaci e pazienti, con funzionalità di alert e richieste automatiche al medico.

## 🚀 Funzionalità principali

- **📋 Gestione pazienti**
  - Aggiungi/modifica/elimina profili pazienti
  - Visualizzazione riepilogativa con conteggio farmaci associati

- **💊 Gestione farmaci avanzata**
  - Scheda completa per ogni farmaco (dose, frequenza, note)
  - Sistema di alert per scorte basse con notifiche browser
  - Calcolo automatico delle compresse residue
  - Supporto per immagini dei farmaci

- **📧 Comunicazione con il medico**
  - Richiesta multipla di farmaci via email
  - Raggruppamento automatico per paziente

- **📱 Interfaccia responsive**
  - Adattamento a dispositivi mobili e desktop
  - Design intuitivo con schede interattive

## 📦 Installazione

Nessuna installazione richiesta! L'applicazione funziona direttamente nel browser:

1. Scarica il file `GestFarm.html`
2. Apri il file con qualsiasi browser moderno
3. Inizia a utilizzare l'applicazione (i dati verranno salvati localmente)

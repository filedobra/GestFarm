# 🏥 Gestione Farmaci - Web App

![Anteprima dell'interfaccia](https://i.imgur.com/JfQ9RYW.png)  
*Schermata principale dell'applicazione GestFarm*

**GestFarm** è un'applicazione web per la gestione intelligente dei farmaci, pensata per pazienti e caregiver. Tieni traccia di scorte, dosaggi, orari di assunzione e ricevi avvisi quando è tempo di riordinare i medicinali.

## 🌟 Caratteristiche principali

- 📦 **Gestione inventario**  
  Registra scatole, compresse rimaste e dosaggio
- ⏰ **Avvisi intelligenti**  
  Notifiche quando le scorte stanno per terminare
- 👨‍⚕️ **Associazione pazienti**  
  Assegna farmaci a specifici pazienti
- 📅 **Piano terapeutico**  
  Programma frequenza di assunzione (giornaliera, settimanale, etc.)
- 📧 **Richiesta prescrizioni**  
  Contatta direttamente il medico con un click
